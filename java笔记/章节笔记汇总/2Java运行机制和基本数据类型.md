# java运行机制

## Java执行流程分析

- .java文件   源文件
- 经过javac进行编译
- .class文件  字节码文件
- 使用Java虚拟机运行，最后显示结果

## 什么是编译

1. 有了Java源文件，通过编译器将其编译成JVM可以识别的字节码文件
2. 在该源文件目录下，通过javac编译工具对源文件进行编译
3. 如果程序没有错误，没有任何提示，但在当前目录下会出现一个Hello.class文件，该文件称为字节码文件，也是可以执行的Java程序

## 什么是运行

1. 有了可执行的Java程序（Hello.class字节码文件）
2. 通过运行工具java.exe对字节码文件进行执行，本质上就是.class装在到JVM（Java虚拟机）执行

# Java程序开发注意事项

1. 对修改后的Hello.java源文件需要重新编译，重新生成新的class文件后，再进行执行才能生效。
2. Java源文件以.java为扩展名，源文件的基本组成部分是类（class），如本类中的Hello类
3. Java应用程序的执行入口时main()方法，它有固定的书写格式：public static void main(String[] args){……}
4. Java语言严格区分大小写！
5. Java方法由一条条语句构成，每个语句以；结束
6. 大括号都是成对出现的，不能缺【可以先写大括号再写代码】
7. 一个源文件中最多有一个public类，其他类个数不限
8. 如果源文件中包含public类，则文件名必须按该类名命名！
9. 一个源文件中最多有一个public类，其他类个数不限，也可以将main方法写到非public类中，然后指定运行非public类，这样入口方法就是非public的main方法

# 注释

1.单行注释

```
代码……              //注释内容
//注释内容
```

2.多行注释

```
/*
注释内容
注释内容
注释内容
*/
```

或者使用快捷键CTRL+/

```
//注
//释
//内
//容
```

3.文档注释

注释的内容可以被JDK中的javadoc.exe所解析，生成一套以网页文件形式体现的该程序的说明文档，一般写在类

```
/**
* @author 小明
* @version 1.0
*/
```

@author，@version叫做javadoc标签，如何使用呢，首先打开命令提示符窗口，使用命令

```
javadoc -d 文件夹名 -xx -yy 文件名
```

其中xx和yy表示javadoc标签，除此之外，可以去网上搜索都有什么可以可用的javadoc标签

```
如：javadoc -d d:\\temp -author -version Hello.java
```

# 字符类型使用

字符型可以表示单个字符，字符型时char，char是两个字节（可以存放汉字），多个字符我们用字符串String

字符类型可以存放数字，输出的是对应的ASCII码对应的字符

# 字符型细节

1. 字符常量是用单引号括起来的单个字符
2. Java允许使用转义字符‘\’来将其后的字符转变为特殊字符型常量
3. 在Java中，char的本质是一个整数，在输出时，是Unicode码对应的字符
4. 可以直接给char夫一个整数值，然后输出时，会按照对应的的Unicode字符输出[97]
5. char类型是可以进行运算的，相当于一个整数，以为他都有对应的Unicode码

# 字符本质

1. 字符型存储到计算机中，需要将字符对应的码值找出来，然后
2. 字符编码表
   1. ASCII码表用一个字节表示，一共128个字符，实际上可以表示256个字符
   2. Unicode编码表，固定大小的编码，使用两个字节来表示字符，字母和汉字统一都是占用两个字节，这样浪费空间
   3. utf-8编码表，大小可变，字母使用1字节，汉字使用3字节
   4. gbk可以表示汉字而茄范围广泛，字母使用1字节，汉字2字节
   5. gb2312可以表示汉字，gb2312<gbk
   6. big5码（繁体中文，台湾，香港）# 布尔类型

1. 布尔类型也叫boolean类型，boolean类型数据只允许取值true和false，无null
2. Boolean占1字节
3. boolean类型适用于逻辑运算，一般用于程序流程控制
   1. if条件控制语句
   2. while循环控制语句
   3. do-while
   4. for
